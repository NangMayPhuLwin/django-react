# django-react-auth (Django + React/Vite)

Separated frontend + backend:

- Frontend (Vite + React): `frontend/`
- Backend (Django): `backend/djangosessionauth/`
- Production build output: `backend/djangosessionauth/static/` (generated by `npm run build`)

## Prereqs

- Node.js + npm
- Python (use your `backend/.venv` or any venv)

## Development

Backend (Django):

```powershell
cd backend/djangosessionauth
python manage.py migrate
python manage.py runserver 8000
```

Frontend (Vite dev server):

```powershell
cd frontend
npm install
npm run dev
```

Notes:

- Vite proxies `/api/*` to `http://127.0.0.1:8000` (see `frontend/vite.config.js`).
- The frontend should call Django APIs using `/api/...` paths.

## Production build (served by Django)

Build the frontend into Django static:

```powershell
cd frontend
npm run build
```

Then run Django and open `http://127.0.0.1:8000/`:

```powershell
cd backend/djangosessionauth
python manage.py runserver 8000
```

## Create users

Create an admin user:

```powershell
cd backend/djangosessionauth
python manage.py createsuperuser
```

Create a normal user (custom management command):

```powershell
cd backend/djangosessionauth
python manage.py create_user --username testuser --password testpass123 --email test@example.com
```
